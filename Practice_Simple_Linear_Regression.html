Step 3

Derive by hand the ordinary least squares regression coefficients and for ECV as a function of social group size.

# remove missing values
d_mod <- d |> 
  drop_na(ECV, Group_size)

# slope (beta1)
b1 <- cor(d_mod$ECV, d_mod$Group_size) * 
      sd(d_mod$ECV) / sd(d_mod$Group_size)

# intercept (beta0)
b0 <- mean(d_mod$ECV) - b1 * mean(d_mod$Group_size)

Q: Do your regression coefficients differ among groups? How might you determine this?

A: The regression coefficients differ among groups. To test if the differences are significant, I can permute the group labels and compare the observed differences in slopes to a null distribution.

Step 6

For your first regression of ECV on social group size, calculate the standard error for the slope coefficient, the 95% CI, and the p value associated with this coefficient by hand. Also extract this same information from the results of running the lm() function.

# define alpha and degrees of freedom for the regression
alpha <- 0.05
n <- nrow(d_mod)
df <- n - 2

# residuals
residuals <- d_mod$ECV - (b0 + b1 * d_mod$Group_size)

# variance
SSE <- sum(residuals^2)
MSE <- SSE / df

# variation in x
SSX <- sum((d_mod$Group_size - mean(d_mod$Group_size))^2)

# SE
SE_b1 <- sqrt(MSE / SSX)

# 95% CI
CI_b1 <- b1 + c(-1, 1) * qt(0.975, df = df) * SE_b1

# p-value
t_b1 <- b1 / SE_b1
p_b1 <- 2 * pt(abs(t_b1), df = df, lower.tail = FALSE)

Step 7

Use a permutation approach with 1000 permutations to generate a null sampling distribution for the slope coefficient. You can use either the quantile method (i.e., using quantiles from the actual permutation-based null sampling distribution) or a theory-based method (i.e., using the standard deviation of the permutation-based null sampling distribution as the estimate of the standard error, along with a normal or t distribution), or both, to calculate this p value.

set.seed(1)
nperm <- 1000
perm <- numeric(nperm)
for (i in 1:nperm) {
  perm.sample <- d_mod
  perm.sample$Group_size <- sample(perm.sample$Group_size)
  perm[i] <- lm(ECV ~ Group_size, data = perm.sample) |>
    tidy() |>
    filter(term == "Group_size") |>
    pull(estimate)
}

# plot
hist(perm,
     main = "Permutation distribution of slope",
     xlab = "Slope",
     breaks = 30)

abline(v = b1, col = "black", lwd = 3)
# permutation SE
perm_se <- sd(perm)

# permutation p-value
p_perm <- (1 + sum(abs(perm) >= abs(b1))) / (nperm + 1)

# theory-based p-value
p_perm_theory <- 2 * pt(abs(b1) / perm_se, df = df, lower.tail = FALSE)

p_perm
[1] 0.000999001
p_perm_theory
[1] 1.146908e-08

Q: What is it that you need to permute? What is the p value associated with your original slope coefficient?

A: I permute the predictor variable (Group_size) relative to the response (ECV) to break their association. The permutation p-value is approximately 0 (p < 0.001), indicating the slope is highly significant.

Step 8

Use bootstrapping to generate a 95% CI for your estimate of the slope coefficient using both the quantile method and the theory-based method (i.e., using the standard deviation of the bootstrapped sampling distribution as an estimate of the standard error).

set.seed(1)
nboot <- 1000
boot <- numeric(nboot)
for (i in 1:nboot) {
  boot.sample <- d_mod |> sample_n(nrow(d_mod), replace = TRUE)
  boot[i] <- lm(ECV ~ Group_size, data = boot.sample) |>
    tidy() |>
    filter(term == "Group_size") |>
    pull(estimate)
}

# parameters
alpha <- 0.05
p.lower <- alpha / 2
p.upper <- 1 - alpha / 2

# plot bootstrap distribution
